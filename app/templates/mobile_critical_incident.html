Use of Force ›

Required Actions

  • Render medical aid and secure scene immediately
  • Notify watch commander as soon as practical
  • Preserve all evidence of force used
  • Obtain witness names and contact information
  • Document all injuries to all parties

Required Paperwork

  • MCPD Stat Sheet
  • Use of Force Report
  • Voluntary Statement (involved officer)
  • Medical documentation if injury occurred
  • DD Form 1920 Alcohol Incident Report (if applicable)

Reminders

  • Supervisor notification required before end of shift
  • Photograph all injuries from all angles
  • Preserve body cam and dash cam footage immediately
  • Involved officer must provide written statement

Domestic Violence ›

Required Actions

  • Separate parties immediately upon arrival
  • Check for visible injuries on both parties
  • Determine primary aggressor — do not defer to "mutual combat"
  • Advise victim of rights and resources available
  • Notify watch commander

Required Paperwork

  • MCPD Stat Sheet
  • Domestic Violence Supplemental Checklist
  • Voluntary Statement (victim and suspect separately)
  • Medical documentation if injury present

Reminders

  • Photograph all injuries and the scene
  • Document presence of children even if not witnesses
  • Mandatory reporting requirements apply
  • Check NCIC/command database for prior DV history

DUI / Alcohol / Drugs ›

Required Actions

  • Conduct standardized field sobriety test if certified
  • Administer breathalyzer if available and legal basis exists
  • Document all observations (odor, slurred speech, unsteady gait)
  • Notify watch commander before arrest
  • Arrange tow for vehicle if driver is incapacitated

Required Paperwork

  • MCPD Stat Sheet
  • DD Form 1920 Alcohol Incident Report
  • Field Sobriety Test documentation
  • Breathalyzer result documentation
  • Voluntary Statement (if subject agrees)

Reminders

  • Photograph vehicle position and any damage
  • Note time of stop vs. time of test — document gap
  • Preserve any open containers as evidence
  • Check for prescription medications that may interact

Serious Accident ›

Required Actions

  • Call EMS immediately if any injury suspected
  • Secure and preserve the scene — no vehicles moved until documented
  • Request accident reconstruction team if fatality or serious injury
  • Notify watch commander immediately
  • Conduct alcohol screening on involved drivers

Required Paperwork

  • MCPD Stat Sheet
  • Traffic Accident Report
  • DD Form 1920 Alcohol Incident Report (if applicable)
  • Voluntary Statement (all drivers and witnesses)
  • Medical documentation for all injured parties

Reminders

  • Photograph scene before any movement occurs
  • Measure skid marks and point of impact
  • Document speed limits, signals, and road conditions
  • Collect insurance and registration from all drivers

Arrest ›

Required Actions

  • Advise subject of rights (Miranda) before custodial interrogation
  • Search incident to arrest — document all items found
  • Notify watch commander of arrest
  • Process subject through correct confinement procedures
  • Notify family/command if required by SOP

Required Paperwork

  • MCPD Stat Sheet
  • Arrest Report / Incident Report
  • Evidence / Property Receipt
  • Voluntary Statement (if subject agrees)
  • Confinement documentation

Reminders

  • Photograph any visible injuries before processing
  • Inventory all property removed — obtain signature if possible
  • Document time of arrest and time of Miranda advisement separately

Search / Consent ›

Required Actions

  • Obtain clear, voluntary, uncoerced consent before search
  • Inform subject they have the right to refuse
  • Limit search to scope of consent granted
  • Document all items found and their exact location
  • Notify watch commander if controlled substances or weapons found

Required Paperwork

  • MCPD Stat Sheet
  • Consent to Search Form (if available)
  • Evidence / Property Receipt (if items seized)
  • Voluntary Statement (document consent in narrative)

Reminders

  • Photograph evidence in place before collection
  • Two-officer witness preferred if available
  • Document exact words used to obtain consent

Evidence Seizure ›

Required Actions

  • Photograph evidence in original position before touching
  • Document exact location, condition, and description
  • Package and seal evidence using proper procedures
  • Maintain chain of custody from point of collection
  • Turn evidence into evidence custodian as soon as practical

Required Paperwork

  • MCPD Stat Sheet
  • Evidence / Property Receipt (itemized)
  • Chain of Custody documentation
  • Incident Report listing seized items

Reminders

  • Every item gets its own evidence bag and label
  • Biological evidence requires biohazard packaging
  • Digital devices — do not power on; place in Faraday bag if available
  • Request supervisor sign-off on high-value or unusual evidence
